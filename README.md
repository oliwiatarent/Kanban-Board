# Aplikacja tablicy Kanban w C++

### Opis projektu
Projekt polegał na zaprojektowaniu i zaimplementowaniu aplikacji desktopowej
realizującej koncept „tablicy kanban”. Aplikacja ma na celu wspomaganie zarządzania
zadaniami, często w ramach procesu zarządzania projektami, ale nie tylko. System ma
obsługiwać zadania podzielone na trzy kolumny, takie jak Do zrobienia (To Do), W trakcie
(Doing), Zrobione (Done).

### Technologie
- C++17
- Qt
